# Glossary

Domain-specific terms used throughout this codebase.

<!-- Add terms in alphabetical order. -->

| Term         | Definition                                                                                           |
| ------------ | ---------------------------------------------------------------------------------------------------- |
| app          | A deployable application or service that lives under `apps/` and consumes shared packages            |
| package      | A reusable library that lives under `packages/` and is consumed by apps or other packages            |
| root         | The workspace root — the top-level directory containing the workspace `package.json`                 |
| workspace    | The entire monorepo managed as a single unit by {{workspaceTool}}                                    |
