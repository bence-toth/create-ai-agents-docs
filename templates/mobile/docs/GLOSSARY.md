# Glossary

Domain-specific terms used throughout this codebase.

<!-- Add terms in alphabetical order. -->

| Term           | Definition                                                                                 |
| -------------- | ------------------------------------------------------------------------------------------ |
| bridge         | The layer that allows JavaScript code to call native (iOS/Android) APIs                    |
| emulator       | A software simulation of an Android device used for development and testing                |
| native module  | A module written in Swift/Kotlin/Objective-C/Java that exposes device APIs to JavaScript  |
| simulator      | A software simulation of an iOS device used for development and testing                    |
