# Glossary

Domain-specific terms used throughout this codebase.

<!-- Add terms in alphabetical order. -->

| Term           | Definition                                                                                          |
| -------------- | --------------------------------------------------------------------------------------------------- |
| breaking change | Any change that requires consumers to update their code (removed export, changed signature, etc.)  |
| consumer       | A project that installs and uses this library as a dependency                                       |
| public API     | Everything exported from the main entry point — the contract with consumers                        |
